Welcome to LoRaWAN Packet Encoder/Decoder API

API Name Description Documentation

/MessageType

 Type of packet

POST Request URL

http://localhost:8080/MessageType

POST Request JSON Parameter

PHYPayload : LoRa PHYPayload (base64 string)

POST Response JSON Data

MessageType : Message Type (UTF-8 string)

/JoinRequest

 Encode Join Request packet with the given parameters

POST Request URL

http://localhost:8080/JoinRequest

POST Request JSON Parameter

DevEUI : Device EUI (hex string)

AppEUI : Application EUI (hex string)

AppKey : Application Key (hex string)

DevNonce : Device Nonce (hex string)

POST Response JSON Data

PHYPayload : LoRa PHYPayload (base64 string)

/JoinAccept

 Decode Join Accept packet with the given parameters

POST Request URL

http://localhost:8080/JoinAccept

POST Request JSON Parameter

PHYPayload : LoRa PHYPayload (base64 string)

AppKey : Application Key (hex string)

DevNonce : Device Nonce "same as JoinRequest" (hex string)

POST Response JSON Data

DevAddr : Device Address (hex string)

NwkSKey : Network Session Key (hex string)

AppSKey : Application Session Key (hex string)

/UnconfirmedDataUp

 Encode Unconfirmed Data Up packet with the given parameters

POST Request URL

http://localhost:8080/UnconfirmedDataUp

POST Request JSON Parameter

DevAddr : Device Address (hex string)

FCnt : Frame Counter (number from 1 to 65535)

FPort : Frame Port (number from 1 to 255)

payload : User data payload (hex string)

NwkSKey : Network Session Key (hex string)

AppSKey : Application Session Key (hex string)

POST Response JSON Data

PHYPayload : LoRa PHYPayload (base64 string)
